# 🌤️ Weather App - React Dashboard

Un'applicazione meteo professionale ispirata a design moderni tipo "Bento Grid".

## 🎯 Obiettivi

- Implementare una gestione complessa della griglia (CSS Grid).
- Integrare API esterne per dati meteo in tempo reale.
- Gestire stati asincroni e loading state.
- Creare grafici dinamici personalizzati senza librerie pesanti.

## 🛠 Tech Stack

- React (Vite)
- CSS Grid/Flexbox (Custom)
- WeatherAPI (o OpenWeatherMap)
- JavaScript ES6+
